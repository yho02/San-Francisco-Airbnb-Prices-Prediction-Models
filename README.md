# San-Francisco-Airbnb-Prices-Prediction-Models

---

## Description
This project analyzes the Airbnb dataset from 2024–2025 to identify key drivers for Airbnb properties in San Francisco. Some of the most influential factors include location, property types, and seasonal trends.

## Features

### Feature Engineering
* **Data Cleaning:** Handled missing and invalid values.
* **New Features:** Constructed variables such as distance to the closest BART station or major attractions.
* **Outlier Removal:** Excluded outliers that skewed property prices to ensure model accuracy.

### Predictive Models
We implemented and compared the following models:
* Linear Regression
* Random Forest
* Gradient Boosting

## Data Source
The data for this project is sourced from [Inside Airbnb](https://insideairbnb.com/).
